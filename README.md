# sopa

A lightweight, fast scripting language written in **Rust**.

## ✨ Features

* 🚀 Fast and lightweight
* 📖 Simple and easy-to-learn syntax
* 🦀 Written in Rust

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/AbolfazlPourisa/sopa.git
cd sopa
```

Run the project:

```bash
cargo run
```

## 📌 Status

> **sopa** is currently under active development. Features and syntax may change as the project evolves.
